# VSCode + Cortex-Debug Template Project

This Project is a template for Debuging with ST-Link and J-Link directlly, not using OpenOCD win Windows.

I prefer to debug STM32F103RB (Nucleo-F103) as an example here, so we should install two STM tools:
- [STM32CubeIDE](https://www.st.com/STM32CubeIDE) - for Jlink/STLink GDB Server and CubeProgramer
 
  
- [STM32CubeMX](https://www.st.com/STM32CubeMX)  - to generate a Makefile project
